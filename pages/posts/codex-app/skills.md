# Codex App 스킬 8선

2026년 4월 22일 기준으로, Codex App 화면에서 `권장`과 `시스템` 영역에 보이는 스킬 8개를 정리했습니다.

## 권장

- **Sora**: OpenAI의 AI 영상 생성 도구입니다. 짧은 설명이나 시나리오를 바탕으로 영상을 만들고 편집하는 흐름에 맞춰져 있습니다.
- **PDF**: PDF 문서를 읽고 만들고 검토하는 데 쓰는 스킬입니다. 페이지 단위 검토와 레이아웃 확인이 필요한 작업에 적합합니다.
- **Playwright**: 실제 브라우저를 자동 제어하는 스킬입니다. 페이지 탐색, 입력, 캡처, UI 테스트 같은 흐름을 다룰 때 유용합니다.

## 시스템

- **Image Gen**: 이미지를 생성하거나 편집할 때 쓰는 기본 스킬입니다. 썸네일, 목업, 간단한 시각 자료 작업에 잘 맞습니다.
- **OpenAI Docs**: OpenAI 공식 문서를 찾아보고 제품과 API 정보를 정리할 때 쓰는 문서 참조 스킬입니다.
- **Plugin Creator**: Codex 플러그인의 기본 구조를 만들고 마켓플레이스용 초기 구성을 잡을 때 쓰는 스킬입니다.
- **Skill Creator**: 새 스킬을 만들거나 기존 스킬 문서를 수정할 때 쓰는 제작형 스킬입니다.
- **Skill Installer**: 준비된 스킬을 설치하거나 외부 스킬을 가져와 붙일 때 쓰는 설치 보조 스킬입니다.
